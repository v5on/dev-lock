DevLock.js Protected

A lightweight, robust JavaScript library to prevent Developer Tools access, disable right-click, and protect your web content.

Try it now: Attempt to press F12, Ctrl+Shift+I, or Right Click on this page.

Simply add this single line to your HTML <head> tag:

<script disable-devtool-auto src="https://cdn.jsdelivr.net/gh/v5on/dev-lock@main/dev-lock.js"></script>

If you want more control, do not use the disable-devtool-auto attribute and initialize manually:

<script src="https://cdn.jsdelivr.net/gh/v5on/dev-lock@main/dev-lock.js"></script>
<script>
    DevLock.init({
        disableRightClick: true,
        redirectUrl: "https://google.com", // Optional
        onDetect: function() {
            alert("Security Alert!");
        }
    });
</script>
